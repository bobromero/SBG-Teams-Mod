# SBG-Teams-Mod
A SBG mod that brings team gameplay to the game! Race in teams to see who can finish the hole first!

Server owners can set player's teams in match settings.

Open Match settings and change the gamemode to Teams.

Each team will have 1 member spawn with the ball and the rest are able to hit their team ball

Not joining a team will automatically place you in a random team when the game starts.

If someone joins mid match they are placed in spectators and can only rejoin when the game ends.

Non mod owners can join and spectate but can't play

Instructions:
1. start game with mod installed
2. change game mode to teams
3. drag players into teams or leave in players section for random teams
4. start game and enjoy!
